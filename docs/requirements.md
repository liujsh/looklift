# looklift 产品需求文档

> 本文档是产品的"宪法":定位、用户、最终形态、路线图。每轮迭代前先对照/更新本文档。
> 敲定于 2026-07-16(v0.2 发布后),与作者 brainstorm 确认。
> 文档约定:每个迭代开工前在 `docs/specs/` 写一份该迭代的 spec(详细设计+验收),
> [design.md](design.md) 只记录已实现的架构实况。

## 一句话定位

**looklift 是给摄影爱好者的开源 AI 调色逆向工具**——看到喜欢的照片,扔进来,
它告诉你这是怎么调的,并给你一个能直接用的 Lightroom 预设。

把「调色审美」变成「可复用、可学习、可分享的资产」。

**对外定位语**(竞品分析结论,见 [competitors.md](competitors.md)):
> LUT 工具给你黑盒,looklift 给你白盒——参数可改、原理可学、预设可攒。100% 本地运行,照片不离开你的电脑。

市场空白:参考图取色流派(Color.io 等)只出黑盒 LUT,风格学习流派(Imagen/Aftershoot/Neurapix)
只卖"省时间"——**没有人做"教学讲解"**,「懂」是我们独占的位置。

## 目标用户(按优先级)

| 画像 | 描述 | 需要什么 |
|---|---|---|
| **P1 入门爱好者**(最大人群,作者本人) | 会用 LR 但不懂参数原理,看到好片不知道怎么学 | GUI、拖拽、中文讲解、一键出预设——工具即老师 |
| **P2 进阶玩家** | 有明确风格追求,在乎还原度 | 批量、refine 精修闭环、风格库管理 |
| **P3 技术型用户** | 会命令行,要自动化 | CLI(已有)、可脚本化、CI 可跑 |

## 核心价值三条线(都做,按节奏推进)

1. **像(还原度)**——套预设后与目标成片的相似度,从六七分做到九分。
   技术抓手:本地近似渲染预览、refine 自动闭环、还原度评分。
2. **库(风格资产)**——风格收藏、检索、标签、批量分析、聚类、分享。
3. **懂(调色教学)**——报告即调色课:参数为什么这么调、经典风格拆解、预置 look 库。

## 最终形态(v1.0)

**功能齐全的本地 GUI 应用**:
- 拖入照片 → 实时看分析过程 → 风格报告(讲解+参数+曲线) → 一键导出预设/sidecar
- 风格库面板:收藏、搜索、套用、分享
- 形态(2026-07-16 敲定):pywebview 独立窗口(像正经桌面软件),`--browser` 浏览器模式兜底
- 打包成 Windows exe(不会装 Python 的人双击即用),CLI 与 GUI 共享同一核心
- 开源(MIT),面向摄影社区发布

## AI 访问策略(关键产品决策)

非技术摄影用户的最大门槛是 AI 额度。方案:**多供应商体系,逐步支持**:

| Provider | 说明 | 排期 |
|---|---|---|
| 本地 Claude Code CLI | 走 CC 订阅额度,零配置(已装即用) | ✅ v0.1 |
| Anthropic 官方 API | BYOK,结构化输出最稳 | ✅ v0.1 |
| OpenAI 兼容中转/供应商 | 国内用户主流方式;自定义 base_url + key + 模型名 | 🎯 计划中 |
| 本地部署视觉模型 | Ollama / vLLM(Qwen-VL 等),免额度的终极方案 | 🎯 计划中 |

设计要求:provider 抽象层,新增供应商不动业务逻辑;GUI 首次启动引导配置。

## 迭代路线图

> 排序逻辑(与作者确认):先把还原度做硬(v0.3)→ 尽早上 GUI 壳收集反馈(v0.4)→
> 之后功能都长在 GUI 里慢慢改。

| 版本 | 主题 | 核心交付 | 验收 |
|---|---|---|---|
| v0.3 | **像:精度闭环** | `preview` 本地近似渲染;`refine --auto` 自动迭代至收敛;还原度评分;`.cube` LUT 导出 | 对同一目标图,auto-refine 3 轮后相似度可量化提升;LUT 在达芬奇可加载 |
| v0.4 | **GUI alpha** | 本地界面:拖拽→分析→报告→导出预设;风格库列表;强度滑杆、before/after 对比条;复用全部核心逻辑 | 不碰命令行完成 U1/U4/U8 全流程 |
| v0.5 | **供应商+库** | OpenAI 兼容 provider、Ollama provider;GUI 内配置;批量分析 | 用非 Anthropic 供应商跑通 analyze |
| v0.6 | **懂:教学深化** | 报告升级"调色课";预置 10 个经典 look;(视情况)风格聚类 | 入门用户看完报告能说出"为什么" |
| v0.7 | **分发** | PyInstaller 打包 exe;首次启动引导;使用文档/视频 | 无 Python 环境的机器双击可用 |
| v1.0 | **发布** | 打磨、修 bug、摄影社区发布(少数派/小红书/即刻) | 有真实外部用户反馈 |

## 非目标

- 不做云服务/账号体系(保持本地工具,零运营成本)
- v1.0 打包只做 Windows exe;**macOS 保持核心兼容**(纯 Python 核心 + pywebview 天然跨平台,
  CI 矩阵含 macOS 保证不退化),Mac 用户走 `pip install looklift`;
  .app 打包与苹果签名($99/年,与零成本定位冲突)v1.0 后视社区需求再议
- 不做局部调整(蒙版/渐变)的视觉逆向——不可靠,以文字讲解代替
- 不追求与 LR 渲染管线像素级一致——preview 是"方向正确的近似"
- v1.0 前不做 Capture One / DxO 等其他软件格式
