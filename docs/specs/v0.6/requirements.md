## 一句话目标

把报告从「这是什么参数」升级成「这是一堂调色课」——每个非零参数配上「为什么这么调」的讲解,
并附 10 个开箱即用的经典风格库,让入门用户不仅套上效果,还能学会原理。

## 验收标准

- [ ] `ANALYSIS_SCHEMA` 扩展后,新分析出的模版包含参数级「为什么」讲解
- [ ] 旧模版(无 rationale 字段)打开报告不报错,优雅降级为 v0.2 原有展现形式
- [ ] 预置 10 个经典风格 look(胶片/电影感/日系等各具代表性),每个都有模版 JSON + 预设 .xmp + 拆解课 HTML 报告
- [ ] 预置 look 库随 `pip install looklift` 分发(package data),`pip install -e .` 开发环境和已安装的正式环境都能定位到
- [ ] 10 个预置 look 的参数与讲解经作者本人 review 确认不对标任何具体商业预设包的参数表(版权边界,见非目标)
- [ ] 现有 pytest 全部回归通过
- [ ] （人工验收,对应 roadmap 验收标准)找入门用户读报告,能复述出至少一个「为什么这么调」

## 非目标

- 不做聚类算法的深度优化;若 v0.5 未裁剪聚类,本迭代不重复实现(见依赖表)
- 不追求 rationale 讲解的学术严谨性,以"入门摄影爱好者能听懂"为标准,不是色彩科学论文
- **不逐一复刻商业预设包(VSCO、知名摄影师预设等)的具体数值**——10 个预置 look 基于通用摄影风格特征
  (如"日系"= 高曝光 + 低对比度 + 青绿阴影染色的通用手法描述)由作者独立调试确定,不对标、不逆向任何具体商业产品的参数表
- 不做预置 look 之外的风格库社区分享功能(backlog U22)
- 不改变现有 `ANALYSIS_SCHEMA` 已有字段的语义,只新增可选字段
