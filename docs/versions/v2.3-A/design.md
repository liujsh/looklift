# v2.3-A 本地文件夹图库设计

## 状态归属

Python 引擎拥有图库 SQLite 数据、扫描与缩略图生成；React 只消费分页查询结果和显式操作结果。原文件路径始终归用户文件系统所有，数据库只保存索引元数据。

## 接口边界

在 sidecar API 增加图库根、扫描任务、分页项目、标签和 Explorer 操作接口。每个项目使用稳定文件身份（规范化路径、大小、修改时间与可选内容指纹）去重；扫描结果必须可重复写入。

## 扫描与缩略图

扫描在后台任务中按目录逐项写入，前端轮询任务进度。缩略图以受控尺寸存入应用数据目录；源文件缺失时保留记录并标示失效，刷新后可恢复。RAW 不在本阶段解码，优先使用可用嵌入预览，否则显示文件占位卡。

## Studio 连接

从图库打开时，若存在最近正式会话则恢复该会话；否则以文件路径创建新会话。图库不会把临时 AI 预览记录为正式版本。

## 停止条件

接口、文件所有权和任务生命周期无矛盾即停止；精确缩略图编码和数据库索引细节由测试固定。
