# v2.3-A 本地文件夹图库需求

## 目标

让用户把 Windows 本地文件夹加入 LookLift 图库，在不复制、不移动原文件的前提下浏览照片、搜索标签，并从图库恢复已有 Studio 会话。

## 范围

- 管理多个 NTFS 索引根；手动添加、刷新和移除索引根。
- 为 JPEG/PNG/WebP 和 RAW 文件建立文件记录、缩略图、基础拍摄/文件信息与用户标签。
- 按名称、路径和标签搜索；展示当前版本及导出记录摘要。
- 从图库项目打开既有会话，或以原文件创建 Studio 会话。
- Windows Explorer 打开文件位置；回收站操作只由系统确认，不在应用内直接删除。

## 非目标

- 不复制或迁移原始照片，不做云同步、NAS 优化或十万级索引。
- 不实现设备/存储卡导入；该能力属于 v2.3-B。
- 不在本阶段承诺 RAW 高位深解码；RAW 仅作为真实文件索引和后续 RAW 门的输入。

## 验收

- 用户可添加含数百张照片的本地目录并看到可用网格。
- 刷新不会重复创建文件记录，移除根不删除原文件。
- 标签和关键词过滤可组合，结果可打开 Studio 或 Explorer。
- 数据库迁移、索引和缩略图测试完全离线。
