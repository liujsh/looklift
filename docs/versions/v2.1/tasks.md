## T6 按需 AI 精修

- [x] 普通消息固定单次调用；每次点击“AI 精修”只追加一轮，累计最多两轮。
- [x] 支持达成、无变更、取消、超限停止和逐轮进度。
- [x] 每轮以最新临时预览为输入，但不自动写正式版本。
- 验收：mock 下轮数和停止原因可确定断言，不存在后台无限调用。

- [x] 建立 schema version、会话、消息、版本和当前正式版本指针的最小表。
- [x] 正式消息与版本使用事务写入，临时预览不入库。
- [x] 增加迁移前最近三份轮换备份和只读恢复入口。
- [x] 覆盖重启恢复、事务回滚、迁移和临时预览不恢复测试。
- 验收：崩溃或重启后只恢复最后确认状态，不出现半版本。
